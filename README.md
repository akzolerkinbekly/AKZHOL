# TechStore - Веб-портфолио жобасы

## 📋 Жоба сипаттамасы
TechStore - заманауи электроника дүкенінің толыққанды веб-сайты. Бұл жоба 5 негізгі тапсырманы қамтиды:

### 1-тапсырма: Анимацияланған карточка тізімі
- 4 өнім карточкасы
- Карточка: hover кезінде translateY(-8px) (0.3s ease)
- Сурет аймағы: hover кезінде scale(1.04)
- «Себетке» батырмасында bounce анимациясы
- Карточкалар кезекпен пайда болады (delay 0.1s, 0.2s...)

### 2-тапсырма: Модаль терезесі
- «Тіркелу» батырмасы модаль ашады
- Ашылу: scale(0.85) + opacity:0 → scale(1) + opacity:1 (280ms)
- Жабылу: scale(1) → scale(0.9) + opacity:0 (200ms)
- Escape пернесі және оверлей арқылы жабылу
- Ашық кезде body скроллы бұғатталады

### 3-тапсырма: Форма валидациясы + Toast
- Аты-жөн (міндетті, мин 2 символ)
- Email (regex валидация)
- Құпиясөз (мин 8 символ, кемінде 1 сан)
- Құпиясөзді қайталау (сәйкестік)
- Қате кезінде: қызыл жиек + анимация
- Сәтті тіркелу: жасыл Toast хабарламасы

### 4-тапсырма: FAQ Аккордеон + Прогресс жолақ
- 5 FAQ сұрағы
- Бір уақытта тек бір жауап ашық
- max-height trick арқылы анимация
- Оқу прогресс жолағы (ашылған сұрақтар %)
- localStorage арқылы ашылған сұрақты сақтау

### 5-тапсырма: Барлық компоненттерді біріктіру
- Жоғарғы навигация (smooth scroll)
- Жобалар секциясы: карточкалар + «Толығырақ» модаль
- Байланыс секциясы: форма + Toast
- FAQ секциясы: аккордеон + прогресс
- Скролл кезінде navbar background өзгереді (scrollY > 50)

## 🚀 Қосымша функциялар
- Тіркелу бір рет - localStorage сақталады
- Толық ақпарат модалы - әр тауар туралы
- Анимациялық фон және градиенттер
- Мобильді құрылғыларға бейімделген дизайн

## 🛠 Технологиялар
- HTML5
- CSS3 (Flexbox, Grid, анимациялар, медиа-сұраулар)
- JavaScript (ES6+)
- Font Awesome иконкалары
- LocalStorage
